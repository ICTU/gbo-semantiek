# Bijlage B: Overzicht URI-namespaces

## GBO Semantiek namespaces

| Prefix | Namespace | Beschrijving |
|--------|-----------|-------------|
| `gbo:` | `https://lod.gbo-semantiek.nl/def/` | Ontologie-definities (klassen, eigenschappen) |
| `gbobegrip:` | `https://lod.gbo-semantiek.nl/begrip/` | Begrippenkader (SKOS concepten) |
| `gboid:` | `https://lod.gbo-semantiek.nl/id/` | Instantie-identificaties |
