# GBO Semantiek

> **Gemeentelijk Basisgegeven Objecten – Informatiemodel Semantiek**
> Versie: v0.1 | Status: Concept

---

Welkom bij de documentatie van **GBO Semantiek** — het informatiemodel voor de semantische beschrijving van gemeentelijke basisgegevens.

## Wat is GBO Semantiek?

GBO Semantiek is een open standaard die een **gedeeld begrippenkader en gegevensmodel** biedt voor gemeenten en ketenpartners. Het doel is om interoperabiliteit te bevorderen en datakwaliteit te verhogen bij de uitwisseling van gemeentelijke basisgegevens.

De standaard beschrijft:

- **Begrippen** (definities, relaties, context)
- **Informatieobjecten** (klassen, attributen, relaties)
- **Artefacten** (UML, JSON Schema, RDF/OWL)

## Inspiratie: het GGM

De opzet van dit project is **geïnspireerd op het [Gemeentelijk Gegevensmodel (GGM)](https://github.com/gemeenteshertogenbosch/GGM)** — een open informatiemodel voor gemeentelijke gegevensuitwisseling. GBO Semantiek volgt een vergelijkbare documentatie- en governance-aanpak.
