# Roadmap

## Inleiding

De roadmap geeft een overzicht van de geplande ontwikkelingen van GBO Semantiek. De roadmap is indicatief en wordt bijgesteld op basis van community-feedback en beschikbare capaciteit.

!!! note "Concept"
    GBO Semantiek bevindt zich in de startfase. De roadmap wordt actief bijgehouden naarmate de werkgroep uitbreidt.

---

## Versie v0.1 — Basis (huidig)

**Status:** In ontwikkeling

### Doelen

- [x] Repository-structuur opzetten
- [x] Documentatiesite inrichten (MkDocs Material)
- [x] Versiebeheerbeleid documenteren
- [x] Governance-structuur beschrijven
- [ ] Eerste concept begrippenkader opstellen
- [ ] Aansluiting MIM beschrijven
- [ ] Placeholder-artefacten aanmaken (`v0.1/uml/`, `v0.1/rdf/`, `v0.1/jsonschema/`)

## Toekomstige thema's (backlog)

De volgende thema's zijn geïdentificeerd voor opname in een toekomstige versie, maar zijn nog niet gepland:

| Thema | Beschrijving |
|-------|-------------|
| DCAT-integratie | Beschrijving van datasets conform DCAT-AP-NL |
| API-profielen | Toepassingsprofielen voor REST API's (OpenAPI) |
| UML-generatie | Automatisch genereren van UML vanuit het model |
| Multilingualiteit | Ondersteuning voor Engelse begrippen/labels |
| Tooling | Scripts voor validatie en transformatie van artefacten |
